# Euclidean-Arpegiator
This project aims to develop a free "Euclidean Arpeggiator" for Reaper.
First version will use Reascript based in Lua to define the core of the plugin, implementing a Graphic User Interface developed with ReaImGui to offer an easy interaction with the arpeggiator.
Lately, a JSFX version will be developed to make the plugin work "live".
Will consider to develop a VST in the future.
